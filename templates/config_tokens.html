Como obter a API Key

  1. Acesse o painel do Intelipost
  2. Navegue até Configurações > API
  3. Clique em "Gerar nova chave"
  4. Copie a chave e cole no campo acima
  5. Clique em "Testar Conexão" para verificar
  6. Salve a chave para uso permanente
